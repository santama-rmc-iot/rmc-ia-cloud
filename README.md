# イントロダクション

2018年度の中小企業診断士 三多摩支部 IoT活用研究会のテーマとして、
作成している電流センサーとCTセンサーを活用するためのソフトウェアです。

会員の橋向氏が構築された「ia-cloud」（IoT向けクラウドサービスフレームワーク)を活用させてもらい実装しています。
